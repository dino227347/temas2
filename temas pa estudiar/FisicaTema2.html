📐 1. Introducción

El estudio de los vectores en la Física es de vital importancia, ya que con frecuencia se necesita trabajar con cantidades físicas que tienen propiedades tanto numéricas como direccionales. En este capítulo se explicarán sus principales propiedades, operaciones y aplicaciones con cantidades vectoriales.

🎯 Objetivo principal:
Que el estudiante tenga conocimiento y destreza para la resolución de problemas de una de las herramientas fundamentales de la Física como son los vectores, ya que este capítulo es una herramienta básica de apoyo a temas que se abordarán más adelante.

🔢 2. Cantidades Escalares y Vectoriales

Se pueden describir dos tipos de cantidades: las escalares y las vectoriales.

🔹 Cantidad Escalar

Una cantidad escalar se especifica por completo mediante un valor único con una unidad adecuada y no tiene dirección.

Ejemplos:

  • 📊 Temperatura: 25°C, 37°C
  • 📦 Volumen: 1.5 L, 250 mL
  • ⚖️ Masa: 70 kg, 1.5 g
  • ⏱️ Intervalos de tiempo: 5 s, 2 horas
  • 💰 Energía: 100 J, 50 cal
  • 📏 Distancia: 10 m, 5 km

Fuente: Física de Serway

🔹 Cantidad Vectorial

Una cantidad vectorial se especifica por completo mediante un número y unidades apropiadas más una dirección.

Ejemplos:

  • 🚗 Velocidad: 60 km/h al norte
  • 📍 Desplazamiento: 10 m al este
  • 📐 Fuerza: 50 N hacia la derecha
  • ⚡ Aceleración: 9.8 m/s² hacia abajo
  • 🔄 Torque: 20 N·m en sentido horario
  • 📈 Momento lineal: 5 kg·m/s al norte

Fuente: Física de Serway

📌 Diferencia clave:
• Escalar: Solo tiene magnitud (número + unidad)
• Vectorial: Tiene magnitud, dirección y sentido

➡️ 3. Representación y Elementos de un Vector

Un vector se representa por una flecha y posee los siguientes elementos: módulo, dirección y sentido.

📏 Módulo

Es la longitud o tamaño del segmento medido a cierta escala. El módulo de un vector es siempre positivo o cero.

Se denota como: |V⃗| o simplemente V

Ejemplo: Si una flecha mide 5 cm en un diagrama donde 1 cm = 10 N, el módulo es 50 N.

🧭 Dirección

La dirección del vector es representada por la recta contenida en el vector. Está determinada por el ángulo que forma el vector con otra recta referencial.

Ejemplo: Un vector puede tener dirección de 30° con respecto al eje x positivo.

👉 Sentido

El sentido del vector está representado gráficamente por la punta de la flecha situada en el extremo del vector. Indica hacia qué lado de la línea de acción se dirige el vector.

Ejemplo: Un vector de 50 N hacia la derecha tiene sentido → (positivo en x).

📝 Notación

Un vector se denota de varias formas:

  • V⃗ → con flecha arriba
  • v → en negrita
  • |V⃗| → módulo del vector
  • Vx, Vy → componentes
📌 Ejemplo gráfico:
Un vector V⃗ con módulo 10 unidades, dirección 30° sobre el eje x positivo y sentido hacia la derecha y arriba.
• Módulo: |V⃗| = 10
• Dirección: 30° respecto al eje x
• Sentido: hacia la derecha y arriba (cuadrante I)
• Componentes: Vx = 10·cos30° = 8.66; Vy = 10·sen30° = 5.00

⚖️ 4. Propiedades de los Vectores

4.1 Igualdad entre dos vectores

Dos vectores M⃗ y N⃗ son iguales si tienen la misma magnitud y si apuntan en la misma dirección a lo largo de líneas paralelas, aun cuando tengan distintos puntos de inicio.

📌 Condiciones para la igualdad:
• |M⃗| = |N⃗| (mismos módulos)
• Misma dirección (ángulos iguales)
• Mismo sentido
• No importa el punto de origen

4.2 Vector Negativo

El vector negativo de un vector A⃗ se define como el vector que, cuando se suma a A⃗, da como resultado cero en la suma vectorial.

Los vectores A⃗ y −A⃗ tienen la misma magnitud pero apuntan en sentido opuesto.

A⃗ + (−A⃗) = 0⃗

4.3 Suma de Vectores

La manera más práctica de sumar vectores es dibujando los vectores uno después del otro (método del polígono).

📌 Propiedades de la suma:
• Ley conmutativa: A⃗ + B⃗ = B⃗ + A⃗
• Ley asociativa: A⃗ + (B⃗ + C⃗) = (A⃗ + B⃗) + C⃗
• Elemento neutro: A⃗ + 0⃗ = A⃗

4.4 Resta de Vectores

La resta de dos vectores utiliza la definición del vector negativo:

A⃗ − B⃗ = A⃗ + (−B⃗)

4.5 Multiplicación de un vector por un escalar

Si un vector V⃗ se multiplica por una cantidad escalar k, el resultado kV⃗ será un vector que tiene la misma dirección de V⃗ y su magnitud será k · |V⃗|.

|kV⃗| = |k| · |V⃗|
  • Si k > 0: el vector mantiene el mismo sentido
  • Si k < 0: el vector invierte su sentido

📊 5. Componentes de un Vector

Se llaman componentes de un vector o componentes rectangulares, a todo vector que se puede describir por completo mediante sus componentes en los ejes x e y.

📐 Componente en x

Vx = |V⃗| · cos θ

Donde θ es el ángulo que forma el vector con el eje x positivo.

📐 Componente en y

Vy = |V⃗| · sen θ

Donde θ es el ángulo que forma el vector con el eje x positivo.

|V⃗| = √(Vx² + Vy²)     θ = tan⁻¹(Vy / Vx)
📌 Ejemplo:
Un vector tiene módulo 10 N y forma un ángulo de 30° con el eje x positivo.
• Vx = 10 · cos 30° = 10 · 0.866 = 8.66 N
• Vy = 10 · sen 30° = 10 · 0.500 = 5.00 N
• |V⃗| = √(8.66² + 5.00²) = √(75 + 25) = 10 N ✓

🧭 6. Vectores Unitarios

Los vectores unitarios son vectores sin dimensiones que tienen una magnitud exactamente de una unidad. Generalmente se usan para especificar una dirección conocida y no tienen otro significado físico.

📌 Notación de vectores unitarios:
• î → dirección x positiva
• ĵ → dirección y positiva
• k̂ → dirección z positiva

Un vector A⃗ se puede escribir como:
A⃗ = Ax î + Ay ĵ + Az k̂
📌 Ejemplo:
El vector M⃗ = (3î + 2ĵ) m significa que tiene:
• Componente en x: 3 m (hacia la derecha)
• Componente en y: 2 m (hacia arriba)
• Módulo: |M⃗| = √(3² + 2²) = √13 m

✖️ 7. Producto de Vectores

Se definen dos tipos distintos de productos vectoriales:

  • Producto escalar (punto): tiene como resultado un escalar
  • Producto vectorial (cruz): tiene como resultado un vector

7.1 Producto Escalar (Producto Punto)

El producto escalar de dos vectores se denota como A⃗ · B⃗. Se define mediante la expresión:

A⃗ · B⃗ = |A⃗| · |B⃗| · cos θ

Usando componentes:

A⃗ · B⃗ = AxBx + AyBy + AzBz
📝 Ejemplo resuelto:
Dados A⃗ = î + 3ĵ + 2k̂ y B⃗ = −3î + 2ĵ − k̂

a) Producto escalar:
A⃗ · B⃗ = (1)(−3) + (3)(2) + (2)(−1) = −3 + 6 − 2 = 1

b) Ángulo entre ambos vectores:
|A⃗| = √(1² + 3² + 2²) = √14
|B⃗| = √((−3)² + 2² + (−1)²) = √14
θ = cos⁻¹(1/(√14·√14)) = cos⁻¹(1/14) = 85.9°

7.2 Producto Vectorial (Producto Cruz)

El producto vectorial se denota como A⃗ × B⃗. Resulta en un vector perpendicular tanto a A⃗ como a B⃗.

|A⃗ × B⃗| = |A⃗| · |B⃗| · sen θ

Usando componentes (determinante):

A⃗ × B⃗ = | î   ĵ   k̂ |
         | Ax   Ay   Az |
         | Bx   By   Bz |

Aplicaciones:

  • 📐 Torque: τ⃗ = r⃗ × F⃗
  • 🔄 Momento angular: L⃗ = r⃗ × p⃗
  • ⚡ Fuerza magnética: F⃗ = q(v⃗ × B⃗)

⚖️ 8. Estática

La estática es la parte de la mecánica que estudia los cuerpos en reposo o en equilibrio. Para su estudio se necesita conocer otras cantidades físicas como las fuerzas y los momentos de torsión.

8.1 Cuerpo Rígido

Un cuerpo rígido es un caso particular de un sistema de muchas partículas donde la separación entre cualquier pareja de ellas siempre permanece constante mientras el cuerpo se mueve.

📌 Características del cuerpo rígido:
• No se deforma bajo ninguna interacción con otros cuerpos
• Las distancias entre sus partículas se mantienen constantes
• Modelo útil para estudiar la estática y la dinámica de rotación

8.2 Momento de Torsión o Torque

El momento de torsión (τ) se define como la tendencia a producir un cambio en el movimiento rotacional.

τ⃗ = r⃗ × F⃗     |τ| = r · F · sen θ

Interpretaciones del torque:

  • τ = F · d donde d = r·sen θ (brazo de palanca)
  • τ = F⊥ · r donde F⊥ = F·sen θ (componente perpendicular de la fuerza)

8.2.1 Regla de signos para el momento de torsión

  • Rotación horaria (sentido de las manecillas del reloj) → τ negativo
  • Rotación antihoraria (contrario a las manecillas) → τ positivo

✅ 9. Condiciones de Equilibrio

Para que un cuerpo rígido esté en equilibrio estático se deben cumplir dos requisitos simultáneamente, llamados condiciones de equilibrio.

📌 Primera Condición

"Si un cuerpo se encuentra en equilibrio, entonces la fuerza resultante que actúa sobre él es igual a cero".

ΣF⃗ = 0
F⃗₁ + F⃗₂ + ... + F⃗n = 0

Equilibrio traslacional

📌 Segunda Condición

"Si un cuerpo se encuentra en equilibrio, entonces el momento de fuerza resultante que actúa sobre él es igual a cero".

Στ⃗ = 0
τ⃗₁ + τ⃗₂ + ... + τ⃗n = 0

Equilibrio rotacional

📌 Resumen:
Para que un cuerpo esté en equilibrio estático completo:
• ΣF⃗ = 0 → No hay traslación
• Στ⃗ = 0 → No hay rotación
Ambas condiciones deben cumplirse simultáneamente.

📝 10. Problemas Resueltos

📌 Problema 1:
Un goniómetro mide la amplitud de los segmentos corporales en 80°. Si los segmentos corporales generan fuerzas de 50 y 70 N como muestra la figura, determine la resultante de dichas fuerzas y el ángulo de la resultante con el eje x.

Solución:
Aplicando el teorema de cosenos:
|R⃗| = √(50² + 70² - 2·50·70·cos100°)
|R⃗| = √(2500 + 4900 - 7000·cos100°)
|R⃗| = √(7400 - 7000·(-0.1736))
|R⃗| = √(7400 + 1215.2)
|R⃗| = √8615.2 = 92.82 N

α = sen⁻¹(50·sen100° / 92.82)
α = sen⁻¹(50·0.9848 / 92.82)
α = sen⁻¹(49.24 / 92.82)
α = sen⁻¹(0.5305) = 32°
📌 Problema 2:
Dados los vectores A⃗ = 3î + 10ĵ + 4k̂, B⃗ = −3î + 10ĵ y C⃗ = −3î − 4k̂, hallar:
a) A⃗ + B⃗ + C⃗ y el módulo
b) A⃗ · C⃗
c) A⃗ × C⃗ y el módulo

Solución a)
R⃗ = (3−3−3)î + (10+10+0)ĵ + (4+0−4)k̂
R⃗ = −3î + 20ĵ + 0k̂
|R⃗| = √((−3)² + 20²) = √(9 + 400) = √409 = 20.22

Solución b)
A⃗ · C⃗ = (3)(−3) + (10)(0) + (4)(−4)
A⃗ · C⃗ = −9 + 0 − 16 = −25

Solución c)
A⃗ × C⃗ = | î   ĵ   k̂ | = (10·(−4) − 4·0)î − (3·(−4) − 4·(−3))ĵ + (3·0 − 10·(−3))k̂
         | 3   10   4  |
         | −3   0   −4 |
= (−40 − 0)î − (−12 + 12)ĵ + (0 + 30)k̂
= −40î + 0ĵ + 30k̂
|A⃗ × C⃗| = √((−40)² + 0² + 30²) = √(1600 + 900) = 50
📌 Problema 3:
Un bloque de 120 N es sostenido por dos cuerdas. Calcular las fuerzas de tensión en cada cuerda.

Solución:
Aplicando las condiciones de equilibrio:
ΣFx = 0 → T1cos37° = T2cos53°
ΣFy = 0 → T1sen37° + T2sen53° = 120 N

Resolviendo el sistema:
T1·0.8 = T2·0.6 → T1 = 0.75·T2
0.75·T2·0.6 + T2·0.8 = 120
0.45·T2 + 0.8·T2 = 120
1.25·T2 = 120
T2 = 96 N
T1 = 0.75·96 = 72 N

❓ Mini Prueba - 10 Preguntas Nivel Universitario

Selecciona la respuesta correcta. Haz clic en "Ver respuesta" para comprobar.

1. Un profesor de física desorientado conduce 3.25 km al norte, 4.75 km al oeste y 1.50 km al sur. Calcule la magnitud del desplazamiento resultante.

a) 6.06 km
b) 3.06 km
c) 4.06 km
d) 5.06 km
✅ Respuesta correcta: c) 4.06 km
Desplazamiento neto en y: 3.25 - 1.50 = 1.75 km al norte. Desplazamiento en x: 4.75 km al oeste. |R| = √(4.75² + 1.75²) = √(22.56 + 3.06) = √25.62 = 5.06 km

2. Calcule el ángulo entre estos dos vectores: A⃗ = −2î + 6ĵ y B⃗ = 2î − 3ĵ

a) 167°
b) 166°
c) 160°
d) 165°
✅ Respuesta correcta: a) 167°
A⃗·B⃗ = (-2)(2) + (6)(-3) = -4 - 18 = -22. |A| = √(4+36) = √40 = 6.32; |B| = √(4+9) = √13 = 3.60. cosθ = -22/(6.32·3.60) = -22/22.75 = -0.967. θ = cos⁻¹(-0.967) = 165.2° ≈ 165°

3. Sean A⃗ y B⃗ dos vectores cuyos módulos son 10 y 15 unidades respectivamente. Calcule el módulo de la resultante si están separados un ángulo de 20°.

a) 24.6 unidades
b) 29.6 unidades
c) 20.6 unidades
d) 21.6 unidades
✅ Respuesta correcta: a) 24.6 unidades
R = √(10² + 15² + 2·10·15·cos20°) = √(100 + 225 + 300·0.9397) = √(325 + 281.9) = √606.9 = 24.6

4. En el cubo de la figura cuya arista es K, calcular el módulo de la suma de los tres vectores.

a) √3
b) √5
c) √7
d) √2
✅ Respuesta correcta: a) √3
Los vectores son perpendiculares entre sí. La resultante es √(K² + K² + K²) = K√3. Para K = 1, |R| = √3.

5. El módulo de un vector A⃗ es de 6 N y está sobre el eje x positivo. Un segundo vector B⃗ tiene una magnitud de 4 N y está en el plano xy formando un ángulo de 30° con el eje x positivo. Calcular el producto cruz B⃗ × A⃗.

a) 12k̂
b) 10k̂
c) 5k̂
d) −12k̂
✅ Respuesta correcta: d) −12k̂
A⃗ = 6î, B⃗ = 4cos30°î + 4sen30°ĵ = 3.46î + 2ĵ. B⃗ × A⃗ = (3.46î + 2ĵ) × 6î = 3.46·6(î×î) + 2·6(ĵ×î) = 12(ĵ×î) = 12(−k̂) = −12k̂

6. Un móvil experimenta tres desplazamientos: A⃗ = (15î + 30ĵ + 12k̂) km, B⃗ = (23î + 14ĵ + 5k̂) km, C⃗ = (−13î + 15ĵ) km. Hallar el vector resultante.

a) 25î + 7k̂
b) 25î + 59ĵ + 17k̂
c) 31ĵ + 7k̂
d) 25î + 31ĵ + 7k̂
✅ Respuesta correcta: b) 25î + 59ĵ + 17k̂
R⃗ = A⃗ + B⃗ + C⃗ = (15+23-13)î + (30+14+15)ĵ + (12+5+0)k̂ = 25î + 59ĵ + 17k̂

7. Un automóvil realiza el siguiente recorrido: primero 25 km al sureste, luego 40 km en una dirección 60° al noreste. Determine el vector desplazamiento de ambos recorridos.

a) 50 km
b) 40 km
c) 30 km
d) 60 km
✅ Respuesta correcta: a) 50 km
Primer vector: 25 km al sureste (45°). Segundo: 40 km a 60° al noreste. R = √(25² + 40² + 2·25·40·cos15°) = √(625 + 1600 + 2000·0.9659) = √(2225 + 1931.8) = √4156.8 ≈ 64.5 km

8. Sean los vectores A⃗ = (5î + 3ĵ + k̂), B⃗ = (3î + ĵ + 5k̂), C⃗ = (−3î + ĵ). Calcule: (A⃗ − 3C⃗) · (2B⃗)

a) 5ĵ + 2k̂
b) −10
c) 10
d) 20
✅ Respuesta correcta: b) −10
3C⃗ = −9î + 3ĵ. A⃗ − 3C⃗ = (5+9)î + (3-3)ĵ + k̂ = 14î + 0ĵ + k̂. 2B⃗ = 6î + 2ĵ + 10k̂. Producto escalar = (14)(6) + (0)(2) + (1)(10) = 84 + 0 + 10 = 94

9. Si el módulo del vector M⃗ es m y el módulo del vector N⃗ es el doble de M⃗, determine la resultante de la suma si están separados 90°.

a) m√3
b) m√5
c) m√7
d) 3m
✅ Respuesta correcta: b) m√5
|M| = m, |N| = 2m, θ = 90°. R = √(m² + (2m)² + 2·m·2m·cos90°) = √(m² + 4m² + 0) = m√5

10. La figura muestra una cuerda elástica atada a dos muelas y estirada hasta pasar por un incisivo. El fin de este dispositivo es aplicar una fuerza F al incisivo. Determine la resultante de las fuerzas si la cuerda tiene una tensión de 25 N.

a) 22.5 N
b) 41.9 N
c) 14.5 N
d) 58.9 N
✅ Respuesta correcta: b) 41.9 N
Fuerza resultante = 2·T·cos(θ/2). Para un ángulo de 120° entre las cuerdas: R = 2·25·cos60° = 50·0.5 = 25 N. Para un ángulo de 60°: R = 2·25·cos30° = 50·0.866 = 43.3 N
